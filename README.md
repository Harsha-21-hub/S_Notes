# 📓 S Notes

S Notes is a lightweight, high-performance infinite-canvas note-taking application for Android 12+. Designed as part of the "S Suite" ecosystem, it features a custom "Nothing OS" inspired dot-matrix aesthetic and a custom-engineered drawing engine to provide a seamless, fluid workspace for quick thoughts and sketches.

## ✨ Features

* **Infinite Canvas Engine:** Built on a highly responsive, custom Android `DrawingView` that allows for fluid, unbounded note-taking and sketching.
* **Nothing-Inspired Aesthetics:** Features a custom monochrome themed icon and dot-matrix visual identity that perfectly aligns with modern Android 13+ theming.
* **Persistent State:** Automatically serializes and saves your strokes locally so you never lose your progress between sessions.
* **Privacy First:** S Notes operates entirely offline and locally. It does not collect, transmit, or sell any personal user data.

## 🚀 How to Install

1. Go to the **Releases** section on the right side of this repository.
2. Download the latest `app-release.apk` file to your Android device.
3. Open the file and tap **Install** (you may need to allow "Install from Unknown Sources" in your browser/file manager settings).

## 🔒 Permissions Guide

Because S Notes saves your custom drawings directly to your device, it requires basic file access to function correctly.

### 1. Storage / Files Access
* **Why it's needed:** To save, serialize, and load your custom `.note` files locally on your device's storage.
* **How to grant:** When prompted upon saving or opening the app, allow the requested storage permissions to ensure your notes are saved securely.

## 🛠️ Technical Specs

* **Package:** `com.hesi.snotes`
